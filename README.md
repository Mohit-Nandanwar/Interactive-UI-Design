# Animations & UI Feedback

## How to Run
1. Ensure Tailwind CSS is linked via CDN or compiled into `output.css`.
2. Open `index.html` in any browser.

## Sections Overview
- **Header:** Introduces the demo.
- **Built-in Animations:** Shows Tailwind’s default `spin`, `ping`, `pulse`, and `bounce`.
- **Hover Buttons:** Demonstrates transitions and transforms.
- **Card Animations:** Uses `group-hover` for lift, rotate, and border reveal effects.
- **Notification Badges:** Combines `ping`, `pulse`, and `bounce` for live indicators.
- **Loading & Feedback:** Displays loaders and bounce dots.
- **Fade-in Cards:** Uses Tailwind’s `animate-fadeIn` (custom keyframe added in `output.css`).

Each section demonstrates **interactive UI feedback** using **pure Tailwind utilities**.
